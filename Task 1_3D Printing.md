# Task 1 - 3D Printing


## What is 3D Printing?

3D printing is basically bringing your virtual design to real life. Whatever
you design on a computer, the 3D printer builds it layer by layer into a
physical object you can hold in your hands. I find this fascinating because
it turns imagination into something real.

---

## What is an STL File?

An STL file is the standard file format used in 3D printing. When you design
something in software like Tinkercad or Fusion 360, you export it as an STL
file. This file contains the shape and geometry of your 3D model and is what
the slicer software reads.

---

## What is a Slicer?

A slicer is software that converts your STL file into instructions the 3D
printer can understand. It slices your model into hundreds of thin layers and
generates a path for the printer to follow.

**Key things I learned about slicer settings:**
- **Supports** — Added when parts of the model are floating in air, so they
don't collapse while printing
- **Infill** — Controls how hollow or solid the print is. Lower infill = 
lighter and uses less material
- **Scale** — You can scale down or up the size of your model in the slicer
before printing
- **Layer height** — Thinner layers = smoother finish but takes more time

---

## Filament — PLA

The most commonly used filament for 3D printing is **PLA (Polylactic Acid)**.
I learned that PLA is preferred because:
- It is **cheap** compared to other filaments
- It is easy to print with — doesn't require very high temperatures
- It is biodegradable and eco-friendly
- Works well for most beginner and hobby projects
