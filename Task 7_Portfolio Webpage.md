# Task 7 - Portfolio Webpage


## What is a Portfolio Webpage?

A portfolio webpage is a personal website that showcases who you are,
what you have built and what you are interested in. It is like a digital
resume that developers and engineers use to present their work to the
world. A good portfolio is responsive — meaning it looks good on both
mobile and desktop screens.

---

## What I Built

I built a fully responsive personal portfolio webpage using HTML and CSS.
The page includes:

- **Hero section** — My name, title and key badges
- **About Me** — Short description of who I am
- **Projects** — All major projects I have worked on
- **Achievements** — Competitions won and certifications earned
- **Connect** — Links to my GitHub and social profiles

---

## Technologies Used

| Technology | Purpose |
|---|---|
| HTML | Structure and content of the page |
| CSS | Styling, layout and responsiveness |
| CSS Variables | Consistent theming across the page |
| Flexbox | Responsive layout for all screen sizes |
| CSS Grid | Skills and project card layouts |
| GitHub Pages | Hosting the portfolio online |
